# InAmigos Foundation NGO Webpage

Welcome to the repository of the InAmigos Foundation NGO Webpage made as a part of internship task. This project was developed as a modern, high-performance web portal for InAmigos Foundation as part of my **Web Development Internship**.

## 📌 Project Overview
This website is built with **Astro 6.x** and **Vanilla CSS** to achieve maximum performance, flexibility, and lightweight page loads. The design features a modern **Mastercard-inspired aesthetic** featuring rich dark palettes, dynamic interactive widgets, and seamless user experiences.

### Key Features
1. **Interactive Navigation Bar**: Floating pill layout containing an active route indicator showing the current page. Collapses into a responsive hamburger overlay on mobile viewports.
2. **Staggered Hero Section**: Staggered scroll-reveal entrance animations for header text, subtitle, buttons, and media frame.
3. **Constellation of Initiatives**: Homepage highlights the NGO's 6 key projects (BachpanSala, Udaan, Jeev, Seva, Prakriti, Vikas) mapped using curved dashed orbit arcs and interactive white satellite circular CTAs.
4. **Active Scroll Reveal**: IntersectionObserver-powered slide-up reveals for sections, cards, and text rows to create a smooth, alive feel.
5. **Secure Donation Checkout**: A dedicated, secure `/donate` portal allowing users to choose an amount and allocation cause, which redirects directly to the hosted Razorpay checkout gateway in the same tab.
6. **Unified Media Gallery**: An image highlights grid with interactive categories and zoom micro-animations.

---

## 🚀 Technical Stack
* **Framework**: Astro 6.x (HTML, JS, TS support)
* **Styling**: Vanilla CSS (Global tokens and components)
* **Animations**: CSS Transitions, Keyframe animations, and JavaScript `IntersectionObserver`
* **Integrations**: Razorpay Hosted Gateway Integration

## 🧞 Local Development & Commands

Run all commands from the project root directory:

| Command | Action |
| :--- | :--- |
| `npm install` | Installs project dependencies |
| `npm run dev` | Launches local development server at `http://localhost:4321` |
| `npm run build` | Compiles the production bundle to `./dist/` |
| `npm run preview` | Previews the compiled production bundle locally |
